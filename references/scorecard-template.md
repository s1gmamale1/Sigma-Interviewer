# Candidate scorecard (printable)

Fill one per candidate, **immediately after** the interview, before discussing with anyone. Quote evidence in every row.

```
Candidate: ______________________   Role: ______________________   Date: __________

ENGLISH GATE (required?  Y / N):   PASS / FAIL      note: ____________________

────────────────────────────────────────────────────────────────────────────
DIMENSION                       SCORE (1–5)   EVIDENCE (quote / observation)
────────────────────────────────────────────────────────────────────────────
English                         [   ]         _________________________________
Behaviour / communication       [   ]         _________________________________
Response completeness           [   ]         _________________________________
AI knowledge                    [   ]         _________________________________
Critical thinking               [   ]         _________________________________
Decision-making                 [   ]         _________________________________
Architecture / system design    [   ]         _________________________________
────────────────────────────────────────────────────────────────────────────

CORE QUESTION  ("scalable build, 30 days, solo — realistic & how to start?")     /100
  [ ]/25  human-in-the-loop: AI proposes, human decides
  [ ]/15  feed AI the spec as context; orchestrate AI
  [ ]/15  AI researches real cases / online
  [ ]/15  challenges the idea (theories, scenarios)
  [ ]/15  plans before code (wishlist→roadmap→plan→docs→mapping)
  [ ]/10  deliberate stack + resource allocation
  [ ]/5   development LAST (not front-end-first / one-shot)
  CORE TOTAL: [    ]/100

1. ___________________________          1. ___________________________

3. ___________________________          3. ___________________________

Language reliance (English vs other): _____________________________________

OVERALL: ___/5      FIT:  Hire-track  /  Maybe-trial  /  Pass
Confidence: High / Med / Low      Caveats: _________________________________
Bottom line (2 sentences): ________________________________________________
```

> Reminder: score each dimension **independently** — don't let one strong impression (or smooth English) pull the whole card up. That's the halo effect.
